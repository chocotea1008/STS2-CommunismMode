# File Map

This repository is intentionally small. These are the important files:

- `CommunismMode/CommunismMode.cs`
  Main mod entry point, shared-gold runtime logic, UI patches, and Neow fixes.
- `Properties/AssemblyInfo.cs`
  Assembly metadata and the shipped mod version.
- `CommunismMode.csproj`
  Build definition and game DLL references.
- `mod_manifest.json`
  Mod loader manifest used by Slay the Spire 2.
- `publish.ps1`
  Local build, deploy, and release ZIP packaging script.
- `README.md`
  Project overview, installation steps, and packaging instructions.
- `CHANGELOG.md`
  Human-readable release history.
- `LICENSE`
  Open-source license for the repository.
